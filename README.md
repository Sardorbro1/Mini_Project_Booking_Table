# Universal Restaurant – Table Booking Platform  
**A beautiful, fully functional, 100% working demo project**
## Features
- Real-time table booking (2, 4, 6 seats & VIP rooms)
- Like / Favorites system (heart icon) – saved even after logout
- User profile with editable name, email, phone
- Admin dashboard with analytics & full booking list
- Search & filters (by name, location, cuisine, rating)
- Smooth scrolling (Explore Now button works perfectly)
- All data saved in localStorage (persists between sessions)
- Modern, responsive, premium design (mobile-friendly)
- Date & time picker powered by Flatpickr
- Login / Register system (with tab switch)
- Booking cancellation
- Estimated revenue calculation for admin

## How to Run
1. Create a new folder
2. Create three files inside:
   - `index.html`
   - `style.css`
   - `script.js`
3. Copy the latest code from the previous messages into each file
4. Open `index.html` in any browser (no server needed)
5. Enjoy! Everything works offline

## Technologies Used
- Pure HTML5 + CSS3 + Vanilla JavaScript
- Font Awesome 6 (icons)
- Flatpickr (date & time picker)
- localStorage (data persistence)
- No backend, no frameworks – 100% client-side

## Project Files
- `index.html` – Full structure & modals
- `style.css`  – Premium modern design
- `script.js`  – All logic (booking, auth, favorites, admin, etc.)

## Notes
- All changes (bookings, favorites, profile) are saved automatically
- Refresh or close the browser – your data stays
- Works perfectly on mobile, tablet, and desktop
